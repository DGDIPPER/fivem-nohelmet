**dg_nohelmet** is a simple standalone FiveM script that disables the automatic bike helmet, letting players keep their own hats, masks, and custom headwear.  
Lightweight, plug-and-play, and works on any framework.

Perfect for RP servers that want consistent character appearance while riding bikes.
